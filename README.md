# 澳洲旅遊訂單與會計系統

本網站使用 GitHub Pages 提供靜態網頁，Supabase 提供登入、訂單、退款／取消與會計資料庫。

## GitHub Pages 發布

1. 將 `index.html`、`.nojekyll` 上傳至儲存庫 `main` 分支根目錄。
2. 進入 **Settings → Pages**。
3. 在 **Build and deployment** 選擇 **Deploy from a branch**。
4. Branch 選擇 `main`，資料夾選擇 `/ (root)`，按 **Save**。
5. 等候約 1–3 分鐘後開啟：
   https://jacky0530-spec.github.io/aus-travel-accounting/
